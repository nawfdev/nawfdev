## About

Software engineer building full-stack products end to end — frontend, backend, infrastructure, and the operational plumbing in between. Comfortable moving across TypeScript/Next.js, Go, Python, and MQL5 depending on what a problem actually needs, rather than staying inside one stack.

Most of the work lives at the intersection of **product engineering** and **systems thinking**: booking engines with real-time room state, POS systems that have to survive flaky networks, homelab infrastructure that has to self-heal, and automation bots that run unattended for months. The common thread is shipping things that work in production, not just in a demo.

**Open to:** freelance/contract engineering work, collaboration on interesting full-stack or infra projects, and technical conversations about system design.
